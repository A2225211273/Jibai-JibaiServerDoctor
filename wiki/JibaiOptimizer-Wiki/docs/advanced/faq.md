# 常见问题

## 插件能保证服务器永不卡吗？

不能。任何插件都不能保证服务器永不卡。硬件、地图、玩家行为、其它插件和 Java 参数都会影响性能。

## 为什么 MSPT 是近似值？

为了兼容 Bukkit / Spigot / Paper / Purpur，插件使用调度器采样计算趋势值，不依赖 Paper 专属 API。

## 为什么 clean 没有清理刚掉落的物品？

插件会保护未达到清理年龄的掉落物，避免误删玩家刚掉落的物品。

## 为什么不主动删除动物和村民？

动物、村民、展示框、盔甲架等可能承载玩家财产或建筑内容，默认不主动删除更安全。

## 可以在 Purpur 上使用吗？

可以。插件目标是 Bukkit / Spigot / Paper / Purpur 通用。
