# 安装教程

## 环境要求

- Minecraft 服务端：1.20+
- Java：17+
- 核心：Bukkit / Spigot / Paper / Purpur

## 安装步骤

1. 获取 `JibaiOptimizer-1.0.0.jar`。
2. 将 jar 放入服务器的 `plugins` 文件夹。
3. 启动或重启服务器。
4. 首次启动后，插件会生成 `plugins/JibaiOptimizer/config.yml` 和 `messages.yml`。
5. 控制台出现 `JibaiOptimizer 已启动`、作者 `即白`、邮箱 `jibai0517@gamil.com` 即表示加载成功。

## 验证安装

在控制台或游戏内执行：

```text
/optimizer status
```

如果可以看到 TPS、MSPT、内存、实体和区块信息，说明插件正常运行。
